# portfolio-covid-19-pb

Projeto de análise e visualização de dados públicos da COVID-19 com foco em comunicação de indicadores.

## Problema
Consolidar dados e apresentar a evolução de casos para facilitar leitura analítica.

## Solução
- Coleta e preparação dos dados
- Análise exploratória
- Visualização dos indicadores principais

## Tecnologias
Python · Jupyter Notebook · Pandas · Visualização de Dados

## Resultado
Base estruturada para acompanhamento temporal e geração de insights de saúde pública.
